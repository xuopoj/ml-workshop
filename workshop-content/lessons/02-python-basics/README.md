# Python 基础语法

本课程介绍机器学习所需的 Python 核心语法，不涉及外部库（NumPy、Pandas、PyTorch 等将在后续课程中介绍）。

## 课程内容

### 基础知识
1. **变量与数据类型**
   - int, float, str, bool, None
   - 类型转换

2. **运算符**
   - 算术运算符 (+, -, *, /, //, %, **)
   - 比较运算符 (==, !=, <, >, <=, >=)
   - 逻辑运算符 (and, or, not)

3. **控制流**
   - if / elif / else 语句
   - 三元表达式

4. **循环**
   - for 循环（range, enumerate）
   - while 循环
   - break 和 continue

### 数据结构
5. **列表 (list)**
   - 创建、访问、修改
   - 切片操作
   - 常用方法 (append, insert, remove, pop)

6. **元组 (tuple)**
   - 不可变特性
   - 解包操作

7. **字典 (dict)**
   - 键值对操作
   - 遍历字典
   - get() 方法

8. **集合 (set)**
   - 去重特性
   - 集合运算（并集、交集、差集）

### 进阶语法
9. **函数**
   - 函数定义与调用
   - 参数类型（位置、默认、关键字、可变参数）
   - Lambda 函数
   - map, filter 函数

10. **字符串操作**
    - 常用方法 (lower, upper, strip, split, join)
    - 字符串格式化（f-string, format, %）

11. **列表推导式**
    - 基本语法
    - 条件过滤
    - 字典推导式、集合推导式

12. **文件操作**
    - 读取文件 (read, readlines)
    - 写入文件 (write, append)
    - with 语句

## 练习题

课程包含 5 个实战练习：
1. **FizzBuzz** - 条件判断与循环
2. **回文检测** - 字符串操作
3. **词频统计** - 字典应用
4. **斐波那契数列** - 递归与迭代
5. **质数判断** - 算法实现

## 学习目标

完成本课程后，你将掌握：
- ✅ Python 基础语法和数据类型
- ✅ 流程控制和循环结构
- ✅ 常用数据结构的操作
- ✅ 函数定义和使用
- ✅ 文件读写基本操作

## 学习建议

- **边学边练**: 运行每个代码示例，观察输出
- **修改代码**: 尝试修改参数，理解代码行为
- **完成练习**: 独立完成练习题，巩固知识
- **不求全懂**: 有些概念可以先用起来，后续会加深理解
